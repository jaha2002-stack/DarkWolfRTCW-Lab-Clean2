ПОРЯДОК ЗАПУСКА

1. RUN_RESET_SAFE_NO_DXR.bat, затем полностью закройте игру.
2. RUN_CLEAN_VISUAL_RT_BALANCED.bat.
3. Если FPS всё ещё недостаточен: RUN_CLEAN_VISUAL_RT_PERFORMANCE.bat.
4. Только затем: RUN_CLEAN_VISUAL_RT_MAXFPS.bat.
5. RUN_CLEAN_VISUAL_RT_EXACT_ORIGINAL.bat — эталон старого медленного режима.

Все режимы сохраняют полное разрешение Clean Release. Half-res, pixel blocks,
новый composite и material-lite ветки здесь отсутствуют.
